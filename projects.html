C Graphs Project (CSCA48H3: Introduction to Computer Science II)

  • Independently created a social media simulation as a project using graphs.
  • The simulation contains 'users' and 'companies' whose relationships ('followers' and 'friends') between each other are stored using a graph.
  • The graph's nodes (users/companies) have both direct and indirect edges to represent friends and followers respectively.
  • Recieved a mark of 94% on the project.

  • The idea is similar to a quadtree, but with regions breaking into two smaller regions recursively instead of four.
  • Recieved a mark of 90% on the project.
